Various Input Parameters

argv[1]:input file name containing root files
argv[2]:Input options: 0: SIM -> DIGI, 1 : DIGI -> RECO (MC), 2: DIGI->RECO (Data)
argv[3]:Least Count of TDC- 0.1
argv[4]: Collated input file name *.root
argv[5]: Print Level- 1000
argv[6]:CMVD Flag: 0: dont include CMVD files, 1: Use it
argv[7]:Mag Field: 0: Use Straightline Fit; 1: Field is there
argv[8]: Trackfit Flag: 0:(Old)SwimSwimmer,1: (New)Kohlahal
argv[9]:Gdml Option: 0:- One stack center & No CMVD, 1:- Two Stacks & No CMVD, 2: Two Stacks with CMVD

System requirements

In order to compile this example, the build essentials must be installed.

  • Debian derivates
    sudo apt install build-essential
  • RHEL derivates
    sudo dnf install make cmake gcc gcc-c++

Building the code

  1. Generate the build environment
    cmake -S <path-to-source-directory> -B <path-to-build-directory>
  2. Start compilation
    cmake --build <path-to-build-directory>
  3. The binary can now be found here
    <path-to-build-directory>/DoxygenSample/DoxygenSample
